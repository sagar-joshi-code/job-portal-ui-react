# JobBoard UI - React Dashboard

A professional, high-fidelity job listing dashboard built with React. This project demonstrates clean UI design, reusable component architecture, and dynamic data handling using Props.

## 🚀 Features
- **Reusable Component Architecture**: A single `Card` component that handles diverse job data.
- **Dynamic Props**: Efficiently rendering job titles, salaries, logos, and status tags.
- **Responsive Flexbox Design**: A custom CSS layout focusing on alignment and spacing for a professional user experience.
- **Modern Iconography**: Integrated `remixicon-react` for a polished look.

## 🛠️ Tech Stack
- **Library:** React.js
- **Build Tool:** Vite
- **Styling:** Plain CSS3 (Flexbox focus)
- **Icons:** Remix Icons
